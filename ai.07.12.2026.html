AI Vendor Digest

Jul 8 – Jul 12, 2026

Model releases & capability updates

xAI (now SpaceXAI)
Grok 4.5 launches, the company's first release since going public and acquiring Cursor [source]
SpaceXAI (formerly xAI) released Grok 4.5 on July 8, calling it an 'Opus-class' model for coding, agentic tasks, and knowledge work, trained alongside newly acquired coding startup Cursor. It's priced at $2/$6 per million input/output tokens with configurable reasoning effort, live today in Grok Build, all Cursor plans, and the SpaceXAI console, though not yet available in the EU.
Why it matters: The Cursor integration and coding-agent framing raise the bar in a category Copilot Studio and GitHub Copilot both compete in, and the EU unavailability is a data-residency wrinkle worth flagging to European D365 customers weighing multi-model agent strategies.

Pricing / API / availability changes

Microsoft
Forms Copilot reaches worldwide general availability [source]
Microsoft made Forms Copilot generally available worldwide on July 8 for customers with qualifying Microsoft 365 Copilot subscriptions. A Copilot button in Forms now lets users generate and refine surveys and quizzes from natural language, review questions, adjust settings, and analyze submitted responses through chat.
Why it matters: A small but concrete example of Copilot moving from preview to GA inside a core Microsoft 365 app that D365/Power Platform customers already use daily — a low-friction talking point in Copilot adoption conversations.
OpenAI
GPT-5.6 goes public in three tiers — Sol, Terra, Luna — with new per-token pricing [source]
On July 9, OpenAI lifted the trusted-partner restriction on GPT-5.6 and made it broadly available in ChatGPT, Codex, and the API. Sol (flagship), Terra (mid-tier), and Luna (fast/cheap) are priced at $5/$30, $2.50/$15, and $1/$6 per million input/output tokens respectively, with Sol claimed to be substantially more token-efficient on coding tasks than prior models.
Why it matters: This is the pricing tier D365/Power Platform partners would plug into Copilot Studio or Azure OpenAI-backed agents once available there — worth comparing against Azure OpenAI Service list pricing as it rolls out.
Perplexity
Agent API adds GPT-5.6 and Grok 4.5; Pro Search reaches GA for Sonar Pro [source]
Perplexity's Agent API added first-party token pricing for OpenAI's GPT-5.6 (Sol/Terra/Luna) and xAI's Grok 4.5, and Pro Search — automated multi-step web search and URL-fetching reasoning — became generally available for Sonar Pro.
Why it matters: Lower direct D365 relevance, but a useful signal of how quickly newly launched frontier models get re-sold through downstream agent platforms — the same distribution pattern Copilot Studio's model picker follows.

Product & platform launches

Microsoft
Foundry pilots an Agent ROI dashboard for measuring AI agent business value [source]
Microsoft began a private preview on July 8 of an AI agent ROI dashboard inside Microsoft Foundry that connects agent costs to task completion, time savings, and business value, letting teams compare agent versions and trace weak performance. Microsoft's roadmap points to granular cost-per-agent-run reporting tied to Azure billing later this year.
Why it matters: Direct hit for D365/Power Platform partners: ROI reporting was the top customer ask via the Copilot Customer Connection program, and per-agent Azure billing chargebacks would materially change how agent costs get justified and sold internally.
OpenAI
ChatGPT Work and GPT-Live voice models launch alongside GPT-5.6 [source]
Alongside GPT-5.6, OpenAI released ChatGPT Work, an agent built on Codex and ChatGPT that acts across a user's apps and files and can stay on a project for hours, plus GPT-Live-1 and GPT-Live-1 mini, a new generation of voice models aimed at more natural spoken conversations.
Why it matters: ChatGPT Work is a direct competitor to Copilot's own 'agent that finishes a job' pitch and to Anthropic's Cowork — the multi-hour autonomous-agent category is now crowded across every major vendor.
Anthropic
Claude Reflect gives users a usage-analytics dashboard, in beta [source]
Anthropic launched Claude Reflect in beta on July 9 at Settings > Reflect, showing Free, Pro, and Max users with memory enabled a monthly recap of the topics they spent time on, their most active day and peak hour, and observations about how they work with Claude.
Why it matters: A consumer-facing feature rather than an enterprise one, but the usage-analytics pattern mirrors what Foundry's new Agent ROI dashboard does for agents — worth watching if Anthropic extends anything similar to Claude for Work/Enterprise admins.
